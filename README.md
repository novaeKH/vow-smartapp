# VOW — Vibe of Words

VOW (Vibe of Words) — это командная словесная игра для устройств Сбера.

Игроки объясняют слова своей команде, не называя само слово и однокоренные формы. Игра поддерживает два режима: обычные слова и молодёжный сленг.

Проект разработан как Canvas App с голосовым управлением через Sber SmartCode.

## Возможности

- командная игра для двух команд: Синие и Красные;
- два режима слов:
  - обычные слова;
  - молодёжный сленг;
- управление голосом, кнопками и касанием экрана;
- таймер раунда на 60 секунд;
- подсчёт очков команд;
- смена команды после завершения хода;
- 4 основных раунда;
- дополнительный 5-й раунд при равном счёте;
- финальное окно с победителем и кнопкой новой игры;
- команда помощи для пользователя.

## Примеры голосовых команд

- «Начать игру»
- «Угадали»
- «Пропустить»
- «Сменить команду»
- «Покажи счёт»
- «Начать заново»
- «Помощь»

## Установка и запуск frontend

Перед запуском должен быть установлен Node.js.

npm install
npm start

После запуска приложение откроется локально:

http://localhost:3000

## Переменные окружения

Для локального запуска с ассистентом используется файл .env.

Пример:

REACT_APP_SMARTAPP=VOW
REACT_APP_TOKEN=your_token_here

В репозиторий не следует добавлять реальный .env с токенами. Для примера можно использовать sample.env.

## Сборка frontend

Для создания production-сборки:

npm run build

После выполнения команды появится папка:

build/

Эту папку можно использовать для загрузки frontend в Sber Studio в разделе хостинга frontend.

## SmartCode-приложение

Голосовое приложение находится в папке:

scenario-example-fixed/

Основной файл сценария:

scenario-example-fixed/src/entryPoint.sc

В нём описаны голосовые команды:

- запуск игры;
- засчитывание слова;
- пропуск слова;
- смена команды;
- показ счёта;
- перезапуск игры;
- помощь.

Файл обработчиков действий:

scenario-example-fixed/src/js/actions.js

Через него SmartCode отправляет команды во frontend-приложение.

## Игровая логика

Матч состоит из 4 раундов.

В каждом раунде сначала играет команда Синие, затем команда Красные.

За каждое угаданное слово команда получает 1 очко.

Сложное слово можно пропустить без начисления очка.

Если после 4 раундов счёт равный, запускается дополнительный 5-й раунд.

После завершения матча приложение показывает победителя и итоговый счёт.

## Режимы игры

### Обычные слова

Классический режим с обычными словами для объяснения.

### Молодёжный сленг

Режим со словами современного молодёжного сленга: например, «кринж», «вайб», «тильт», «краш» и другие.
